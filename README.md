### 📌 Objetivos del Taller

El presente taller tiene como finalidad que los estudiantes logren:

- Comprender y resolver ejercicios prácticos.
- Trabajar de manera colaborativa en grupos.
- Utilizar un entorno de desarrollo (IDE) a elección.
- Aplicar conceptos básicos de programación en Python 3.
- Comprender y aplicar conceptos fundamentales de Programación Orientada a Objetos (POO).
- Utilizar herramientas básicas de control de versiones con Git y GitHub (creación de repositorios y subida de archivos).

---

## 🛠️ Tecnologías a utilizar

- Python 3
- Git
- GitHub
- IDE a elección (por ejemplo: VS Code, PyCharm, etc.)
